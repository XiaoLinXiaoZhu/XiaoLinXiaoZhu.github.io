# 显示器

背景

之前就一直好奇hdr开启之后能带来什么样的效果，但是一直没有机会体验，然后今天晚上想要捣鼓一下，于是乎认认真真看了几篇教程……

总结来说实际上就是一句话，hrd确实能够在显示hdr内容的时候 带来更高的亮度和更高的对比度，但是在显示sdr内容的时候，hdr反而会使其不能够正常显示，这尤其是在一些较为老的软件上，往往会出现一些问题。

而且，当hdr的最高亮度等级不到400的时候实际上，hdr显示出来的内容和sdr的内容并没有太大的区别，如果想要追求更显著的效果，恐怕还要将最大亮度拉高到1000nit以上才能够获得较为明显的差异效果。
