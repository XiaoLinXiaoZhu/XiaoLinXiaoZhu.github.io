Microsoft Edge 中的键盘快捷方式

Edge for Windows

键盘快捷方式是指按键或按键组合，可提供另一种方式来执行通常使用鼠标执行的操作。 下面是适用于 新版 Microsoft Edge 和 Microsoft Edge for Mac 的列表。

  因为我打算对于我个人使用网页浏览器的效率进行提升，故在这里记录一下我最近学到的几个比较好用的快捷键。

总览

然后下面是我认为会常用到的：

最常用的

  • Ctrl + D
    将当前选项卡另存为收藏夹
  • Ctrl + Shift + D
    在新文件夹中将所有打开的标签页另存为收藏夹
  • Alt + Shift + B
    将焦点放在收藏夹栏中的第一项上
  • Ctrl + F
    在页面上打开查找
  • Ctrl + Shift + Delete
    打开清除浏览数据选项
    -Ctrl + Shift + E
    在边栏中打开搜索

使用以提高效率的

  • Ctrl + H
    打开历史记录
  • Ctrl + J
    打开“下载”文件夹
  • Ctrl + Shift + O
    打开收藏夹
  • Ctrl + Shift + Y
    打开集合

标签页操作

  • 操作页面

    • Ctrl + M

      • 将当前标签页设为静音（切换）

    • Ctrl + P

      • 打印当前页

    • Ctrl + S

      • 保存当前页

    • Ctrl + R

      • 重新加载当前页

    • Ctrl + Shift + R

      • 重新加载当前页，忽略缓存内容

    • Esc

      • 停止加载页面；关闭对话框或弹出窗口

  • 关闭页面

    • Ctrl + N

      • 打开新窗口

    • Ctrl + W

      • 关闭当前标签页

    • Ctrl + Shift + W

      • 关闭当前窗口

  • 新建页面

    • Ctrl + Shift + N

      • 打开新的 InPrivate 窗口

    • Ctrl + Shift + T

      • 重新打开上一个关闭的标签页并切换到该标签页

    • Ctrl + T

      • 打开新标签页并切换到该标签页

    • Ctrl + Shift + K

      • 复制当前标签页

  • 切换页面

    • Ctrl + 1, 2, … 8

      • 切换到特定标签页

    • Ctrl + 9

      • 切换到最后一个标签页

    • Ctrl + Tab

      • 切换到下一个标签页
