    判断向量组的解的情况

    • 方程组

      • 不齐次Ax = β

        • 只有一组非零解

    • 齐次方程组

      • 等价

        • Ax=0有非0解

        • A的列向量组成线性相关

        • r(A)<n

        • 当m=n的时候

          • |A| = 0

      • 等价

        • Ax=0无非0解

        • A的列向量组成线性无关

        • r(A)=n

        • 当m=n的时候

          • |A| != 0

    • 非齐次线性方程组

      • 只有唯一解

        • 等价

          • β能由A的列向量表示且表示式唯一

          • r(A,β) = r(A) = n

          • n=m时

            • |A| != 0

      • 有无穷解

        • β能由A的列向量表示且表示式不唯一

        • r(A,β) = r(A) < n

        • n=m时

          • |A| = 0

      • 无解

        • β不能由A的列向量表示

        • r(A,β) = r(A) +1

          • 不能被线性表示

        • n=m时

          • |A| = 0

      • AX = λX

        • X为A的特征向量

          • 特征向量不为0

    • （A-λE ）X=0

      • 实际上就是这个方程有非0解

      • |A-λE| = 0

    • 特征值的乘积 = A的行列式的值
